# 第一天：
## 学习php输出
echo php内置函数用于输出，当然跟print类似，但不一样

`<?php`  
` echo "a";`  
`?>`  

则是输出a

## 学习php注释


`//`  单行注释
`/* 里面是内容 */`   多行注释
  
在注释中的内容是给人看的，机器不识别

## 学习变量

`<?php`  
` $a=1;`  
` echo $a;`  
`?>`  
  
echo为输出，那么在以上代码中我们输出的内容则是1。而不是$a,因为$a的内容已经被我们上面替换成了1，这个1可以理解为是作为数据存储在$a里面的内容。
